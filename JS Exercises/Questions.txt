1. Shopping Cart Total Calculation:
You are developing an e-commerce website and need to calculate the total price of items in a shopping cart. Write a JavaScript function that takes an array of objects representing items in the cart (each object containing a "price" property) and returns the total price. Use both a normal function and an arrow function for this calculation.

2. Temperature Conversion:
Your weather application needs to convert temperatures between Celsius and Fahrenheit. Write two JavaScript functions: one to convert Celsius to Fahrenheit and another to convert Fahrenheit to Celsius. Use both normal and arrow functions for these conversions.

3. User Authentication:
In a user authentication system, you need to validate user credentials. Write a JavaScript function that takes a username and password as arguments and returns true if the user is authenticated, otherwise false. Implement this function using both a normal function and an arrow function.

4. Sorting Algorithm:
You are developing a task management application and need to implement a sorting algorithm to sort tasks based on priority. Write a JavaScript function that takes an array of task objects (each object containing a "priority" property) and returns a sorted array of tasks by priority. Implement this function using both a normal function and an arrow function.

5. Text Processing:
Your text processing application requires a function to count the occurrences of a specific word in a text. Write a JavaScript function that takes a text string and a word to search for, and returns the number of occurrences of that word in the text. Use both a normal function and an arrow function for this task.
